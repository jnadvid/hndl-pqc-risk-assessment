# HNDL / PQC Risk Assessment

**Herramienta de evaluación de riesgo post-cuántico y exposición HNDL**  
*Diseñada y desarrollada por José Israel Nadal Vidal*

## ¿Qué es?

Evalúa la exposición de tu organización al riesgo **Harvest Now, Decrypt Later (HNDL)**
y el grado de preparación para la transición post-cuántica (PQC).

- 12 dominios de riesgo
- 48 preguntas
- Informe ejecutivo PDF con plan de remediación por fases
- Sin dependencias externas — funciona en local

## Uso

### Opción A — Script Python
```bash
python hndl_pqc_server.py
```

### Opción B — Ejecutable Windows
Descarga el `.exe` desde [Releases](../../releases)
